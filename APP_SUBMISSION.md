## Description (한국어)

```
LIVE스코어는 전 세계 스포츠 경기의 실시간 정보를 제공하는 앱입니다.

주요 기능:
• 실시간 스코어 - 축구, 야구, 농구, 배구 등 주요 종목 경기 현황
• 경기 일정 - 날짜별, 리그별 경기 일정 조회
• 경기 상세 정보 - 라인업, 통계, 기록 확인
• 리그 순위 - 주요 리그 팀 순위 및 성적

스포츠 팬이라면 놓칠 수 없는 필수 앱입니다.
```
